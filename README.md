# mern-boilerplate
Essentials only Boilerplate for new Mongo + React + NodeJS Projects

---
## Server (default to localhost:3001)
- Mongoose/MongoDB
- Express + Express Router
- NodeJS v10.x

---
## Client (default to localhost:3000)
- React 16.8.x

---
## Integration
Basic support for:
- TravisCI 
- Heroku
